Pawsitive Futures – UX/UI Website Design

This project focused on designing a responsive website for Pawsitive Futures, a non-profit animal rescue organization. The goal was to create a clear, welcoming, and easy-to-navigate experience that helps users adopt pets, learn about services, and support the organization.

Using Figma, I designed a 3-page MVP (Home, Services, and Pet Care) across mobile, tablet, and desktop breakpoints. I worked on layout structure, visual hierarchy, and component consistency to ensure a smooth user experience on all screen sizes. I also designed reusable UI elements such as navigation bars, cards, buttons, and footers to maintain consistency throughout the site.

From a UX perspective, I focused on accessibility, readability, and user flow, making sure key actions like adoption, volunteering, and learning more were easy to find. The final outcome is a responsive, user-friendly interface that reflects the organization’s values of care, trust, and community involvement.

Personas

I created three user personas to guide the UX design for Pawsitive Futures: prospective adopters, volunteers/foster caregivers, and partners/donors. Each persona highlights key goals, frustrations, and needs to ensure the website supports different user motivations.

These personas helped shape clear navigation, detailed pet profiles, accessible service information, and trust-building elements like testimonials and impact metrics. Designing with these users in mind allowed me to create a more focused, inclusive, and user-centered experience.

Information Architecture & Sitemap

I designed the sitemap to support the needs identified in the personas by organizing content into clear, intuitive sections. Pages like Adopt, Get Involved, Services, and Your Pets were structured to reduce confusion and improve navigation. This ensured users can quickly find what they need, whether they want to adopt, volunteer, donate, or learn more about pet care.

User Journey

I created a primary adopter persona and mapped their user journey to understand how users move from discovering pets to completing an adoption. This helped me identify key pain points such as lack of information, hesitation during comparison, and friction in the adoption process. The journey map directly influenced clearer content structure, stronger calls to action, and features like detailed pet profiles and adoption feedback.

Moodboard

I created the mood board to define the emotional direction of the project, focusing on warmth, compassion, and a sense of home. The imagery highlights real moments between people and pets to build trust and emotional connection, while the warm color palette reinforces care, hope, and approachability. This visual direction helped set the tone before moving into layout and structure.

Low-fidelity wireframes

Using this foundation, I designed low-fidelity wireframes to focus on content hierarchy, navigation, and user flow without visual distraction. The wireframes map out key sections such as adoption, services, and pet care, ensuring users can easily scan information and take action. This step helped validate layout decisions early and ensured the design aligned with both user needs and the brand’s emotional goals
